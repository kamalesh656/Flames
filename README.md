# FLAMES Game 🔥

A simple Java console application that predicts the relationship between two people using the FLAMES algorithm.

## About

FLAMES stands for:

- F – Friends
- L – Love
- A – Attraction
- M – Marriage
- E – Enemy
- S – Sister

The program removes common letters from two names, counts the remaining letters, and uses the FLAMES elimination process to determine the relationship.

## Features

- Accepts two names as input
- Removes common characters
- Calculates remaining letter count
- Implements FLAMES logic using StringBuilder
- Displays the final relationship

## Technologies Used

- Java
- StringBuilder
- Scanner Class
